The pinnacle of GeoFS integration and evolution. This pack bundles over 27 community addons, unified under a single glassmorphic user interface with performance optimizations and a custom draggable panel engine.

## ✨ Features

- **🎯 Glassmorphic UI**: High-end transparency, blur effects, and smooth transitions that rival modern desktop simulators.
- **🎯 Unified Draggable System**: Every panel is draggable, minimizes seamlessly, and remembers its position across sessions.
- **🎯 AI ATC (GPT-Powered)**: Real-time, voice-capable Air Traffic Control that understands your aircraft's state and location.
- **🎯 Murphy's Law (Failures)**: A complex emergency simulation system with procedure checklists for every failure type.

## 🛠️ Installation

### Standalone (Tampermonkey)

1. Install the [Tampermonkey](https://www.tampermonkey.net/) extension.
2. Install the script via [**Greasy Fork**](https://greasyfork.org/en/scripts/000000-geofs-v3-9-addon-pack).
3. Refresh GeoFS and the module will automatically initialize alongside the HUD.
