# V1.0.0 - Initial Nexus Core Release

**Release Date**: April 2026
**Release Status**: GA (General Availability)

## Summary

The inaugural release of the **GeoFS-Nexus Core**, a professional-grade framework for injecting modular enhancements into GeoFS. This version established the foundation for the entire nexus, focusing on a non-destructive injection method.

## Changelog

- **Core Engine**: Introduced the GeoFS-Nexus core script for modular addon loading.
- **Initial Addons**: 
    - Streetlights & Taxiway Lighting
    - Scitor Jobs integration
    - GPWS (Ground Proximity Warning System)
- **Architecture**: Implemented dynamic fetch-and-eval logic to minimize initial page load impact.
- **Organization**: Established the internal `/Resources` directory for asset management.
