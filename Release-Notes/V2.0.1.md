# V2.0.1 - Telemetry & Stability Patch

**Release Date**: April 2026
**Release TAG**: `V2.0.1`

## Summary

A critical stabilization release aimed at resolving severe runtime crashes introduced by scope isolation issues in the Random Jobs and Maritime Structures modules.

## Changelog

- **Stability Fixes**: 
    - Resolved widespread `ReferenceError` crashes in the Random Jobs module by implementing explicit window exports for core job functions.
    - Fixed `markerLayer` initialization failures.
    - Added comprehensive `try-catch` blocks around all modular initializations to prevent a single addon failure from crashing the entire modpack.
- **UI Logic**: 
    - Relocated the Maritime Structures toggle to prevent bottom-bar overlaps.
    - Moved the HUD minimize button to the left edge for better ergonomics.
