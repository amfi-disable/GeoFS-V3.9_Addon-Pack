# V3.0.0 - Localization & Rebranding

**Release Date**: April 2026
**Release TAG**: `V3.0.0`

## Summary

The most significant organizational update since the project's inception. This release marks the official transition to the **GeoFS Modpack** branding and achieves 100% localization of all resource dependencies.

## Changelog

- **Project Rebranding**: 
    - Officially renamed the project from "GeoFS-Modpack" to **GeoFS Modpack**.
    - Updated all internal attributions and README documentation.
- **Full Localization (Massive Sweep)**: 
    - Migrated **all** external script and asset dependencies to internal repository storage.
    - Localized 10,789 livery textures to the user's dedicated fork.
    - Localized all 3D models (Jetbridges, Streetlights, Maritime Structures).
    - Localized all audio assets (GPWS, G-Breathing, Realism sounds).
- **Standalone Synthesis**: 
    - Synchronized the pre-release and released environments for a clean production build.
- **Addon Consolidation**: 
    - Integrated the primary Control Hub (PR #3) for better management of active modules.
