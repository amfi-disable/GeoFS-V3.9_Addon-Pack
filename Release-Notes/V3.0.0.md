# GeoFS-V3.9_Addon-Pack V3.0.0 Official Release

This major release marks the official promotion of the project to a **Stable Release**. It introduces a complete architecture overhaul, a high-fidelity aircraft performance database, and a modernized UI design system.

---

## 🌟 Major Highlights

### ✈️ High-Fidelity Aircraft Performance Database

We have moved away from generic weight/fuel estimations. V3.0.0 ships with a built-in database of **40+ aircraft profiles**, providing real-world:
- **OEW (Operating Empty Weight)**
- **MTOW (Maximum Takeoff Weight)**
- **Max Fuel Capacity**
The system automatically detects your aircraft ID or matches community models by name to apply the correct profile.

### ⛽ Advanced Fuel & Load Management

The new **Fuel Tab** in the HUD allows for precision flight planning:
- **Real-time Consumption**: Burn rates are calculated based on thrust and aircraft-specific profiles.
- **Dynamic Weight**: Your aircraft's mass in GeoFS now changes realistically as you burn fuel or adjust payload.
- **Auto-Refuel**: A new toggle for long-haul testing or leisure flying that keeps your tanks topped up automatically.

### 🖥️ Modernized Glassmorphic HUD

The **Information Display (HUD)** has been rebuilt with a premium aesthetic:
- **Dynamic Telemetry**: IAS, Mach, GS, ALT, AGL, HDG, V/S, AOA, and G-Force.
- **Draggable Interface**: Move the HUD anywhere; positions are saved automatically via localStorage.
- **Official Branding**: The "GeoFS-V3.9_ADDON-PACK" branding is integrated into the HUD header.

### 📁 Infrastructure Promotion

- **Unified Pathing**: The project has migrated all resources to the `RELEASED_FILES` directory.
- **Stable Branding**: Rebranded as **GeoFS-V3.9_Addon-Pack** across all scripts, readme, and documentation.

## 🛠 Workflows

### 📥 1. Installation (New Users)

1. Install the [Tampermonkey extension](https://www.tampermonkey.net/).
2. Create a new script.
3. Copy the code from [**`RELEASED_FILES/tampermonkey.user.js`**](https://github.com/AwesomeOddEven-NightKeys-LunarBlink/03_GeoFS-V3.9_Addon-Pack/blob/main/RELEASED_FILES/tampermonkey.user.js).
4. Save and refresh GeoFS.

### 🔄 2. Migration (Beta/V2.1 Users)

If you were using the old `PRE_RELEASE_FILES` or `test-main.js`:
- Your old script path is now a **404**.
- **Action Required**: You must update your Tampermonkey script code to point to the new `RELEASED_FILES/main.js` path. The V3.0.0 loader handles this automatically.

### 🛠 3. Developer / Contributor Workflow

If you wish to add a new aircraft profile:
1. Locate `const AIRCRAFT_PERFORMANCE` in `RELEASED_FILES/main.js`.
2. Add your entry using the format: `"ID": { name, oew, mtow, fuel }`.
3. If the aircraft is a community model, use a unique lowercase string from its name as the key.
4. Submit a Pull Request.
