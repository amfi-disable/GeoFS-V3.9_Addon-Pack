# V1.1.1 - UI Interface Polish

**Release Date**: April 2026
**Release TAG**: `V1.1.1`

## Summary

A aesthetics-focused update addressing user feedback regarding readability in various lighting conditions and correcting timing regressions in the Jetbridge system.

## Changelog

- **UI Styling**: 
    - Updated all addon dropdown menus to use a Dark Blue theme with white text for maximum contrast.
    - Improved consistency of the "Nexus" cyber-cyan accent colors across all panels.
- **Bugfixes**: 
    - Adjusted Jetbridge animation timing to prevent clipping into certain aircraft models.
    - Fixed out-of-bounds dragging for the HUD panel.
- **Maintenance**: 
    - Updated Tampermonkey metadata to properly point to the GitHub repo for live updates.
