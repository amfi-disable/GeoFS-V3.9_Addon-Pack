# V2.0.0 - The Glassmorphism Overhaul

**Release Date**: April 2026
**Release TAG**: `V2.0.0`

## Summary

A major version jump introducing a completely redesigned User Interface. This release transformed the look and feel of the modpack from a basic menu to a premium, immersive flight monitoring suite.

## Changelog

- **New UI System**: 
    - Introduced **Glassmorphism** styling for all cards and panels.
    - Implemented a modular, draggable HUD system with persistent positioning (saved across sessions).
- **Procedures Integration**: 
    - Added **17 detailed aircraft procedures** (Pre-flight, Take-off, Approach, etc.) for realistic operations.
- **Keybind Overhaul**: 
    - Added `[K]` as the global toggle for the HUD.
    - Added `[Y]` for Slew Mode controls.
    - Standardized Autothrottle keybind to `Shift + ~`.
- **Logic Improvements**: 
    - Added robust null checks for menu initialization to prevent start-up crashes.
